Detailed Description

An entity present in the game world.

An entity with a position, a sprite and an optional collision box. Every entity in the main widget entities vector will be updated each tick (i.e. it's update method will be called). Entities HAVE to be spawned in the game world using createEntity method of the widget class, otherwise, the parent property won't be correctly set.
