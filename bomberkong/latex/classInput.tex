\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classInput}{Input}} class manages and stores the states of every key used in the game. 

This static class tracks the status of every key used in the game. it is updated at every key event in {\ttfamily widget.\+cpp} and can be used to know if a given key is pressed or not. It is also used to bind keys to specific actions (movements, placing a bomb, etc.)
