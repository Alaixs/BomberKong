\doxysubsection{Detailed Description}
An entity present in the game world. 

An entity with a position, a sprite and an optional collision box. Every entity in the main widget {\ttfamily entities} vector will be updated each tick (i.\+e. it\textquotesingle{}s {\ttfamily update} method will be called). Entities HAVE to be spawned in the game world using {\ttfamily create\+Entity} method of the {\ttfamily widget} class, otherwise, the {\ttfamily parent} property won\textquotesingle{}t be correctly set.
